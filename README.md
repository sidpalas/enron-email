# enron-email

### Download the data

website: https://www.cs.cmu.edu/~enron/

```
wget https://www.cs.cmu.edu/~enron/enron_mail_20150507.tar.gz
```

### unzip the data

```
tar xvzf ./enron_mail_20150507.tar.gz.tar.gz
```
